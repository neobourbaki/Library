Let $M$ be a right module over a ring $R$, $N$ a left module over $R$, and $L$ an abelian group. Then a map $f: M \times N \to L$ is $R$-*balanced* if

- $f(m, n + n') = f(m, n) + f(m, n')$
- $f(m + m', n) = f(m, n) + f(m' + n)$
- $f(mr, n) = f(m, rn)$

for all $m, m' \in M$, $n, n' \in N$, and $r \in R$.
